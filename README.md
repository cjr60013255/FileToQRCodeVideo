# FileToQRCodeVideo
Encrypts Files then converts to QR codes then a Single Video File, Doing this in reverse is in progress...

GUI is next after everything works smoothly.

Project requires:
* Python 3.8.8 or Higher pereferably
* pip or pip3
* ffmpeg

Setup steps: (Run these in powershell pereferably)
1. Clone repo "git clone 'this repo'"
2. run "pip3 install cryptography" or "pip install cryptography"
3. run "pip3 install Pillow" or "pip install Pillow"
4. run "winget install ffmpeg"
5. Make sure ffmpeg is defined in the system PATH var (shows up when "ffmpeg --version" is run in powershell)
